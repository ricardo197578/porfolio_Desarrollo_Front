# Portfolio Personal

Este proyecto es un sitio web estatico de portfolio personal desarrollado con HTML y CSS.

## Contenido

- Seccion de presentacion personal.
- Navegacion interna por anclas.
- Tarjetas de proyectos.
- Seccion de peliculas favoritas.
- Formulario basico de contacto.

## Tecnologias

- HTML5
- CSS3

## Ejecucion local

1. Abri una terminal dentro de la carpeta del proyecto.
2. Ejecuta:

```bash
python -m http.server 8000
```

3. Abri en el navegador: `http://localhost:8000`

## Estructura

- `index.html`: estructura principal del sitio.
- `css/styles.css`: estilos del portfolio.
- `AGENTS.md`: guia para agentes de codificacion.
